Q2 : A quoi servent les tags h1, h2 et h3 ?

Ces balises servent à créer des titres dans la page html. Les numéros indiquent l'importance du titre (h1 étant le titre le plus important).

Q3 : A quoi sert le tag p ?

La balise p sert à délimiter un paragraphe.

Q4

Les tags HTML commencent par une balise ouvrante <foo> et finissent par une balise fermante </foo>

Q5 : Comment mettre en gras ou en italique ?

Le tag <b> permet de mettre en gras et le tag <i> permet de mettre en italique.

Q6 : Et comment faire pour mettre un mot à la fois en gras et en italique ?

Il suffit de mettre les 2 balises (en faisant attention de les refermer dans l'ordre inverse d'ouverture).

Q7 : On souhaite que le titre de la fenêtre du navigateur soit "Compte Rendu de TP1", comment faire ?

Il faut inclure le tag <title> dans le tag <head>.

Exercice : Service HTTP

Q1

Le fichier /etc/services contient la liste des services du réseau Internet. Plus précisément, pour chaque service on retrouve le port de communication associé sur la machine, le protocole de transport (couche réseau du modèle OSI) associé et les alias pouvant exister.

Q2

La commande cat /etc/services | grep http indique que le numéro de port TCP pour le service http est 80.
